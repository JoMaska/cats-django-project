    Используя Django, Django Rest Framework и Angular развернул простейший сервис.
    Общение с бекендом только посредством rest api.
    Взял за основу простейшую модель кота.
    На сервисе предусмотрена возможность регистрации пользователя в
    качестве заводчика кота.
    Заводчик имеет возможность создавать кота, удалять кота, редактировать параметры кота(любые). Например имя, возраст, породу и волосатость.
    Соответственно заводчики имеют право видеть и редактировать только своих котов.
    Создал раздел - Чат. В котором предусмотрел отправку сообщений по вебсокету другим заводчикам.
    Для создания интерфейса использовал Angular Material.
    Упаковал получившееся приложение в Docker-контейнер.
